# React CLI

## Usage of the cli

After downloader the hook typescript template,
just execute
```python
python3 cli.py
```
and you will see a interactive console for options

⚠️: Caution, please name the hook as camelCase

## Features
- Form Component
    - Formly
    - React Hook Form
- UI Framework
    - Material UI
    - Zeit UI
    - React Bootstrap
- Generator
    - Hook
    - Global Hook
    - Page
    - Components
